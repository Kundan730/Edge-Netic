# PWA Icons

For a production deployment, you need to create the following icon files:

- `icon-192.png` - 192x192px icon
- `icon-512.png` - 512x512px icon
- `favicon.ico` - Standard favicon

## Quick Generation

You can generate icons using:

1. **Online Tools**:
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator

2. **From SVG**:
   Create an SVG logo and export at required sizes

3. **Design Guidelines**:
   - Use transparent background
   - Keep design simple and recognizable at small sizes
   - Follow PWA icon best practices
   - Use brand colors (cyan #00ffff and purple #a855f7)

## Suggested Design

For Edge-Netic, consider:
- Shield icon with circuit board pattern
- Gradient from cyan to purple
- Dark background
- Tech/cyberpunk aesthetic
